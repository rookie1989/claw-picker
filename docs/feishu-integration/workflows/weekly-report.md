每周五自动汇总本周任务、会议、代码提交，生成结构化周报并一键发送。

## 触发方式

### 自动触发（推荐）

每周五 18:00 自动生成：

```json
{
  "cron": {
    "jobs": [
      {
        "name": "周报生成",
        "schedule": {
          "kind": "cron",
          "cron": "0 18 * * 5"
        },
        "payload": {
          "kind": "agentTurn",
          "message": "生成本周工作周报：汇总本周完成的飞书任务、参与的日历会议、GitHub 代码提交，并创建飞书文档"
        }
      }
    ]
  }
}
```

### 手动触发

```
生成本周工作周报
```

## 周报模板定制

### 研发工程师版

```
生成周报，重点包含：
1. 完成的功能/修复的Bug（附PR链接）
2. 代码提交统计
3. 技术难点和解决方案
4. 下周技术目标
格式：按以上4个模块，每模块 3-5 条要点
```

### 产品经理版

```
生成周报，重点包含：
1. 需求进展（已确认/评审中/待排期）
2. 本周会议和决议
3. 指标变化（如有数据）
4. 下周重点事项
```

### 管理者版（团队周报）

```
生成团队周报，汇总以下成员本周工作：张三、李四、王五
包含：各自完成事项、跨团队协作、风险事项、下周重点
```

## 常见问题

**Q: GitHub 数据读取失败？**
确认已配置 GitHub Token：`openclaw config plugin github --token ghp_xxx`

**Q: 如何修改自动发送时间？**
修改 cron 表达式：`0 17 * * 5`（周五17点）

**Q: 想按季度生成季报？**
```
/cron add "每季度最后一个工作日生成季度总结报告"
```
