# 🌅 晨间简报工作流

每天早上自动推送一份包含日程、天气、邮件摘要的简报。搭建时间约 30 分钟。

## 效果预览

```
☀️ 早安！今日简报 2026-03-30

📅 今日日程（3 个）
09:30  团队站会
14:00  产品评审会
16:00  1:1 沟通

🌤️ 天气  上海  15-22°C，晴

📬 邮件摘要
3 封未读 · 1 封来自老板（项目进度确认）

💡 提醒
- 今天提交周报
- 明天是需求截止日期
```

---

## 前置要求

- OpenClaw 已安装并运行
- 飞书应用已配置（见 [飞书基础配置](../basic/calendar-quickstart.md)）
- 启用技能：`feishu-calendar`、`feishu-task`

---

## 第一步：配置飞书技能

编辑 `~/.openclaw/config.json`：

```json
{
  "plugins": {
    "feishu": {
      "appId": "cli_xxx",
      "appSecret": "xxx",
      "userToken": "u-xxx"
    }
  },
  "skills": {
    "entries": ["feishu-calendar", "feishu-task"]
  }
}
```

---

## 第二步：创建定时任务

```bash
/cron add "每天早上 8 点生成晨间简报，包含今日日程、天气和待办任务"
```

或通过配置文件添加：

```json
{
  "cron": {
    "jobs": [
      {
        "name": "晨间简报",
        "schedule": {
          "kind": "cron",
          "cron": "0 8 * * 1-5"
        },
        "payload": {
          "kind": "agentTurn",
          "message": "生成今日晨间简报：包含今日日程、今日天气（上海）、未读邮件摘要、今日待办任务"
        }
      }
    ]
  }
}
```

---

## 第三步：测试

手动触发验证效果：

```
生成今日晨间简报
```

如果输出格式不符合预期，可以在消息中添加格式要求：

```
生成晨间简报，格式：先列日程，再列天气，最后列待办，每段用 --- 分隔
```

## 进阶：自定义简报内容

### 加入 GitHub 动态

```
生成晨间简报，额外包含：昨日 GitHub 仓库的 PR 和 Issue 摘要
```

### 加入新闻摘要

```
生成晨间简报，额外包含：科技行业今日热点 3 条
```

### 推送到飞书群

在定时任务的 message 中加入目标：

```
生成晨间简报并发送到飞书群「研发团队」
```

---

## 常见问题

**Q: 定时任务没有触发？**
检查网关是否在运行：`openclaw gateway status`

**Q: 日程读取不到？**
确认 `userToken` 有效，或重新授权：`openclaw auth feishu`

**Q: 想调整触发时间？**
修改 cron 表达式：`0 9 * * 1-5`（每天9点）、`30 7 * * *`（每天7:30包含周末）
