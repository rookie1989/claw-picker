### 第二部分：配置步骤（8 分钟）

#### 步骤 1：创建飞书应用（2 分钟）

**画面**：飞书开放平台录屏

**操作步骤**：
1. 访问 open.feishu.cn
2. 登录企业账号
3. 点击"创建企业自建应用"
4. 填写应用信息
5. 添加权限（日历、任务、多维表格、文档）
6. 获取 App ID 和 Secret

**旁白**：
> "首先，我们需要在飞书开放平台创建一个应用。
> 
> 注意，这里需要管理员权限。
> 
> 权限一定要选对，不然功能会用不了。"

#### 步骤 2：获取用户 Token（1 分钟）

**画面**：OAuth 授权流程演示

**操作步骤**：
1. 配置重定向 URL
2. 访问授权页面
3. 点击同意
4. 获取 User Token

**旁白**：
> "User Token 是访问你个人数据的钥匙。
> 
> 一定要妥善保管，不要泄露给别人。"

#### 步骤 3：配置 OpenClaw（3 分钟）

**画面**：配置文件编辑 + 终端操作

**操作步骤**：
```json
{
  "plugins": {
    "entries": ["feishu"],
    "feishu": {
      "appId": "cli_xxx",
      "appSecret": "xxx",
      "userToken": "u-xxx"
    }
  },
  "skills": {
    "entries": [
      "feishu-bitable",
      "feishu-calendar",
      "feishu-task",
      "feishu-create-doc"
    ]
  }
}
```

```bash
# 重启网关
openclaw gateway restart

# 测试连接
/message 今天有什么会议
```

**旁白**：
> "把凭证填入配置文件，记得重启网关。
> 
> 然后用一条简单消息测试，看看是否正常工作。"

#### 步骤 4：创建自动化工作流（2 分钟）

**画面**：配置定时任务

**操作步骤**：
```json
{
  "cron": {
    "jobs": [
      {
        "name": "晨间简报",
        "schedule": {"kind": "cron", "expr": "0 8 * * *"},
        "payload": {"kind": "agentTurn", "message": "生成晨间简报"}
      }
    ]
  }
}
```

**旁白**：
> "配置完成后，可以设置定时任务。
> 
> 比如每天早上 8 点自动发送晨间简报。
> 
> 这样你一到公司就能看到当天的安排。"
