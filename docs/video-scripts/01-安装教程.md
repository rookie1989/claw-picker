## 🔧 第一步：安装 OpenClaw（1:30 - 3:00）

### 画面
- 终端录屏，显示安装命令
- 关键命令高亮显示
- 进度条动画

### 旁白
```
打开你的终端，输入以下命令全局安装 OpenClaw：

npm install -g openclaw

这个命令会从 npm 下载并安装 OpenClaw 到全局。

安装过程中，你会看到一些依赖包的下载进度。
耐心等待，大约需要 1-2 分钟。

安装完成后，输入 openclaw --version 检查是否安装成功。

如果显示版本号，说明安装成功！

比如我这里是 1.0.0 版本。
```

### 屏幕文字
```bash
# 安装命令
npm install -g openclaw

# 验证安装
openclaw --version
```

### 可能的问题提示（屏幕下方小字）
```
💡 如果遇到权限错误，在命令前加 sudo（Mac/Linux）
💡 如果下载慢，可以切换 npm 镜像源
```

## ⚙️ 第二步：初始化配置（3:00 - 5:00）

### 画面
- 终端录屏，显示 onboard 向导
- 每个步骤的选项用方框高亮

### 旁白
```
安装成功后，我们运行初始化向导：

openclaw onboard

这会启动一个交互式的配置向导。

首先，向导会问你如何运行 OpenClaw。
对于大多数用户，选择「本地运行」即可。

接下来，选择你的主要聊天应用。
我用 Telegram 举例，因为它配置最简单。

然后，选择 AI 服务提供商。
这里我选择硅基流动，国内访问速度快，支持多种模型。

输入你的 API Key。
如果还没有，可以访问 siliconflow.cn 注册获取。

接下来选择模型。
推荐使用 Qwen2.5-72B-Instruct，性价比高。

最后，确认配置，向导会自动生成配置文件。
```

### 屏幕文字
```bash
# 初始化向导
openclaw onboard

# 配置选项：
1. 运行方式：本地运行
2. 聊天应用：Telegram
3. AI 服务：硅基流动
4. 模型：Qwen2.5-72B-Instruct
```

### 提示框
```
💡 API Key 获取：
访问 siliconflow.cn → 注册账号 → 控制台 → API Keys
```

## 🔗 第三步：连接聊天应用（5:00 - 7:00）

### 画面
- 分屏显示：左侧终端，右侧 Telegram 界面
- 二维码/链接用放大框显示

### 旁白
```
配置完成后，我们需要连接聊天应用。

以 Telegram 为例：

第一步，在 Telegram 搜索 @BotFather。

第二步，发送 /newbot 创建新机器人。

第三步，按提示输入机器人名称和用户名。

第四步，BotFather 会给你一个 Token。
复制这个 Token，非常重要！

第五步，在终端运行：
openclaw connect telegram

粘贴你的 Bot Token。

第六步，将生成的链接添加到 Telegram 机器人。

完成！现在你的机器人已经连接成功了。
```

### 屏幕文字
```bash
# 连接 Telegram
openclaw connect telegram

# 粘贴 Bot Token
# 添加 webhook 链接到 @BotFather
```

### 提示框
```
💡 其他聊天应用：
- WhatsApp: openclaw connect whatsapp
- Discord: openclaw connect discord
- iMessage: 需要 macOS 设备
```
