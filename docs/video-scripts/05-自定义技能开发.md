#### 步骤 2：编写 SKILL.md（2 分钟）

**画面**：编辑器录屏

```markdown
# weather-query

查询任意城市当前天气。

**当以下情况时使用此技能**：
(1) 用户提到"天气"、"气温"、"下雨"
(2) 用户询问某个城市的天气
(3) 用户说"今天天气怎么样"
```

**旁白**：
> "SKILL.md 告诉系统什么时候使用这个技能。

#### 步骤 3：编写 skill.js（5 分钟）

**画面**：代码编辑 + 讲解

```javascript
const https = require('https');

module.exports = {
  name: 'weather-query',
  
  execute: async (context, params) => {
    // 1. 提取城市名
    const city = extractCity(context.message);
    
    if (!city) {
      return '❌ 请告诉我要查询哪个城市';
    }
    
    // 2. 调用天气 API
    const weather = await getWeather(city);
    
    // 3. 返回结果
    return formatWeather(city, weather);
  }
};

function extractCity(message) {
  const cities = ['北京', '上海', '广州'];
  for (const city of cities) {
    if (message.includes(city)) return city;
  }
  return null;
}

function getWeather(city) {
  return new Promise((resolve, reject) => {
    const url = `https://wttr.in/${city}?format=j1`;
    https.get(url, (res) => {
      let data = '';
      res.on('data', chunk => data += chunk);
      res.on('end', () => resolve(JSON.parse(data)));
    }).on('error', reject);
  });
}

function formatWeather(city, data) {
  return `
🌤️ ${city} 天气

🌡️ 温度：${data.temp}°C
💧 湿度：${data.humidity}%
💨 风速：${data.wind} km/h
  `.trim();
}
```

**旁白**：
> "skill.js 是技能的核心。
> 
> execute 函数是入口，接收上下文参数。
> 
> 返回的内容会发送给用户。
> 
> 这里我们调用了免费的天气 API。"

#### 进阶 3：多轮对话（3 分钟）

**画面**：对话流程演示

```javascript
module.exports = {
  execute: async (context, params) => {
    const state = context.session.state || {};
    
    if (!state.city) {
      // 第一次：询问城市
      context.session.state = { step: 'waiting_city' };
      return '请问要查询哪个城市的天气？';
    }
    
    if (!state.date) {
      // 第二次：询问日期
      context.session.state.city = context.message;
      context.session.state = { step: 'waiting_date', city: context.message };
      return '请问要查询哪天的天气？';
    }
    
    // 第三次：查询并返回
    const weather = await getWeather(state.city, state.date);
    context.session.state = {}; // 清空状态
    return formatWeather(state.city, weather);
  }
};
```

**旁白**：
> "多轮对话让技能更智能。
> 
> 使用 session.state 保存对话状态。
> 
> 每次对话根据状态执行不同逻辑。"
