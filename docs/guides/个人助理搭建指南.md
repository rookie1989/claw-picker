## 🚀 第一步：安装 OpenClaw（10 分钟）

### 1.1 安装 Node.js

```bash
# 检查是否已安装
node -v
npm -v

# 如果未安装，访问 https://nodejs.org 下载安装
```

### 1.2 安装 OpenClaw

```bash
npm install -g openclaw

# 验证安装
openclaw --version
```

### 1.3 初始化配置

```bash
openclaw onboard
```

按向导选择：
1. **运行方式**: 本地运行
2. **聊天应用**: Telegram（最简单）
3. **AI 服务**: 硅基流动
4. **API Key**: 从硅基流动控制台获取
5. **模型**: Qwen2.5-72B-Instruct（推荐）

---

## 🔗 第二步：连接 Telegram（10 分钟）

### 2.1 创建 Telegram 机器人

1. 在 Telegram 搜索 `@BotFather`
2. 发送 `/newbot`
3. 输入机器人名称，如「我的 AI 助理」
4. 输入用户名，如 `my_ai_assistant_bot`
5. 复制 Bot Token（格式：`123456:ABC-DEF1234...`）

### 2.2 配置 Webhook

```bash
# 在终端运行
openclaw connect telegram

# 粘贴 Bot Token
# 复制生成的 webhook 链接
```

### 2.3 设置 Webhook

1. 在 BotFather 输入 `/setwebhook`
2. 选择你的机器人
3. 粘贴 webhook 链接
4. 看到「Webhook saved」表示成功

### 2.4 测试连接

在 Telegram 中找到你的机器人，发送：
```
你好
```

如果收到回复，恭喜！连接成功！

## ⚙️ 第三步：个性化配置（5 分钟）

### 3.1 修改系统提示

编辑配置文件（通常位于 `~/.openclaw/config.json`）：

```json
{
  "systemPrompt": "你是一个高效的个人助理，帮助用户处理日常工作。请：\n1. 回答简洁明了\n2. 提供可执行的建议\n3. 主动询问是否需要进一步帮助\n4. 支持中文和英文"
}
```

### 3.2 设置常用指令

在配置中添加快捷指令：

```json
{
  "commands": {
    "/todo": "帮我列出今天的待办事项",
    "/summary": "请总结以下内容：",
    "/translate": "请将以下内容翻译成英文：",
    "/email": "帮我写一封邮件，主题是："
  }
}
```

### 3.3 配置飞书集成（可选）

如果需要管理飞书任务/日历：

```json
{
  "integrations": {
    "feishu": {
      "enabled": true,
      "appId": "cli_xxx",
      "appSecret": "xxx"
    }
  }
}
```

## 🛡️ 安全建议

### 保护 API Key

- ✅ 不要将 API Key 提交到 GitHub
- ✅ 使用环境变量存储敏感信息
- ✅ 定期更换 API Key

### 数据隐私

- ✅ 敏感信息不要发给 AI
- ✅ 本地运行优先于云服务
- ✅ 定期清理聊天记录

## 🆘 常见问题

### Q: 机器人不回复怎么办？

A: 检查：
1. Webhook 是否正确设置
2. 服务器是否运行中
3. API Key 是否有效
4. 查看日志：`openclaw logs`

### Q: 回复太慢怎么办？

A: 尝试：
1. 切换到更快的模型
2. 检查网络连接
3. 减少消息长度
4. 使用本地部署

### Q: 可以多人使用吗？

A: 可以，但需要：
1. 配置多用户权限
2. 设置不同的系统提示
3. 考虑升级到团队版
