## 🚀 步骤 1：选择部署方案

### 方案 A：飞书企业自建（推荐）

**优势**：
- ✅ 深度集成飞书套件

- ✅ 消息卡片丰富

**要求**：
- 飞书企业账号
- 管理员权限（创建应用）

### 方案 B：钉钉企业自建

**优势**：
- ✅ 钉钉用户覆盖广
- ✅ 审批流程集成

**优势**：

## 🔧 步骤 2：飞书应用配置

### 2.1 创建应用

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 点击"创建企业自建应用"
3. 填写应用信息：
   - 应用名称：团队助手
   - 应用图标：上传团队 Logo
   - 应用描述：团队 AI 协作机器人

### 2.2 配置权限

**必需权限**：
- 发送消息
- 读取群聊
- 日历权限
- 任务权限
- 多维表格权限
- 用户信息

**配置步骤**：
1. 进入"权限管理"
2. 搜索并添加上述权限
3. 点击"申请权限"
4. 管理员审批

### 2.3 配置事件订阅

1. 进入"事件订阅"
2. 启用事件订阅
3. 配置订阅地址：
   ```
   https://your-domain.com/feishu/event
   ```
4. 订阅事件：
   - 接收消息
   - 消息已读
   - 日历事件
   - 任务更新

### 2.4 获取凭证

记录以下信息：
- App ID（cli_xxx）
- App Secret
- Encrypt Key
- Verification Token

## 🛠️ 步骤 3：OpenClaw 配置

### 3.1 基础配置

编辑 `~/.openclaw/config.json`：

```json
{
  "plugins": {
    "entries": ["feishu"],
    "feishu": {
      "appId": "cli_xxx",
      "appSecret": "xxx",
      "encryptKey": "xxx",
      "verificationToken": "xxx",
      "userToken": "u-xxx"
    }
  },
  
  "model": {
    "default": "qwen3.5-plus"
  },
  
  "skills": {
    "entries": [
      "feishu-bitable",
      "feishu-calendar",
      "feishu-task",
      "feishu-create-doc"

```

## 📝 步骤 4：创建团队知识库

### 4.1 创建多维表格

使用飞书多维表格创建项目跟踪表：

| 字段 | 类型 | 说明 |
|------|------|------|
| 任务名称 | 文本 | 任务标题 |
| 负责人 | 人员 | 分配给谁 |
| 状态 | 单选 | 待办/进行中/已完成 |
| 优先级 | 单选 | 高/中/低 |
| 截止日期 | 日期 | 截止时间 |
| 进度 | 数字 | 0-100% |

### 4.2 配置自动化

在多维表格中设置自动化规则：

1. 当"状态"变为"已完成"时：
   - 发送通知到群聊
   - 更新项目进度

2. 当"截止日期"为明天时：
   - 提醒负责人

## 🚀 步骤 6：高级功能

### 6.1 晨间站会

**时间**：每个工作日 9:30

**自动发送**：
```
☀️ 早安！晨间站会

📅 今日重点
- 项目 A：前端联调
- 项目 B：需求评审

👥 成员状态
- 张三：进行中 (80%)
- 李四：待开始
- 王五：已完成

💡 今日目标
完成项目 A 前端开发
```

**配置**：
```json
{
  "cron": {
    "jobs": [
      {
        "name": "晨间站会",
        "schedule": {"kind": "cron", "expr": "30 9 * * 1-5"},
        "payload": {"kind": "agentTurn", "message": "发送晨间站会"}
      }
    ]
  }
}
```

### 6.2 日报自动汇总

**时间**：每个工作日 18:00

**流程**：
1. 收集团员日报消息
2. AI 汇总整理
3. 创建飞书文档
4. 发送到群聊

### 6.3 项目周报

**内容**：

- 代码提交统计
- 问题与风险

## 🐛 常见问题

### Q: 机器人无响应？

**A:** 检查：
1. 网关是否运行
2. 事件订阅配置
3. 权限是否充足
4. 消息格式是否正确

### Q: 如何邀请机器人进群？

**A:** 
1. 飞书开放平台 → 应用 → 版本管理与发布
2. 点击"发布"
3. 在飞书中搜索机器人
4. 邀请进群

### Q: 如何保护敏感信息？

**A:** 
```json
{
  "security": {
    "allowlist": ["ou_xxx1", "ou_xxx2"],
    "blockSensitive": true
  }
}
```
