## 🎯 什么是 OpenClaw？

OpenClaw 是一个**自托管的多通道网关**，让你可以在 WhatsApp、Telegram、Discord 等聊天应用中随时随地与 AI 助手对话。

**核心优势：**
- 📱 在任何聊天应用中使用 AI
- 🔒 数据完全可控，自托管
- 🔌 支持多个 AI 服务（硅基流动、OpenAI 等）
- 🤖 支持多智能体协作

## 准备工作

### ✅ 系统要求

| 组件 | 要求 | 检查命令 |
|------|------|----------|
| Node.js | >= 18 | `node -v` |
| npm | >= 9 | `npm -v` |
| 网络 | 稳定连接 | - |
| 聊天应用 | WhatsApp/Telegram/Discord 等 | - |
| AI 服务 | 硅基流动/OpenAI 等 | - |

### 📝 检查步骤

**1. 检查 Node.js 版本**

```bash
node -v
```

如果版本低于 18，请前往 [nodejs.org](https://nodejs.org) 下载最新版本。

**2. 检查 npm**

```bash
npm -v
```

**3. 准备 AI 服务账号**

推荐使用**硅基流动**（国内访问速度快）：

1. 访问 [siliconflow.cn](https://siliconflow.cn)
2. 注册账号
3. 进入控制台 → API Keys
4. 创建新的 API Key 并保存

> 💡 **提示**: API Key 只显示一次，请妥善保存！

## 安装 OpenClaw

### 步骤 1: 全局安装

打开终端，运行：

```bash
npm install -g openclaw
```

**安装过程：**
- 会从 npm 下载 OpenClaw 及依赖
- 大约需要 1-2 分钟
- 安装完成后无错误提示即成功

### 步骤 2: 验证安装

```bash
openclaw --version
```

**成功示例输出：**
```
openclaw/1.0.0 darwin-arm64 node-v22.22.0
```

### ⚠️ 可能遇到的问题

**问题 1: 权限错误**

```
Error: EACCES: permission denied
```

**解决方案**: 在命令前加 `sudo`（Mac/Linux）

```bash
sudo npm install -g openclaw
```

**问题 2: 下载速度慢**

**解决方案**: 切换 npm 镜像源

```bash
# 使用淘宝镜像
npm config set registry https://registry.npmmirror.com

# 重新安装
npm install -g openclaw
```

## 初始化配置

### 步骤 1: 运行向导

```bash
openclaw onboard
```

这会启动交互式配置向导。

### 步骤 2: 选择运行方式

```
? 如何运行 OpenClaw?
❯ 本地运行 (推荐)
  Docker 容器
  系统服务
```

选择 **本地运行**（适合大多数用户）。

### 步骤 3: 选择聊天应用

```
? 选择主要聊天应用
❯ Telegram (最简单)
  WhatsApp
  Discord
  iMessage (需要 macOS)
  其他
```

**推荐选择 Telegram**，配置最简单。

### 步骤 4: 选择 AI 服务

```
? 选择 AI 服务提供商
❯ 硅基流动 (推荐，国内速度快)
  OpenAI
  Anthropic
  其他
```

### 步骤 5: 输入 API Key

粘贴你之前在硅基流动控制台创建的 API Key。

### 步骤 6: 选择模型

```
? 选择模型
❯ Qwen2.5-72B-Instruct (推荐，性价比高)
  Qwen2.5-32B-Instruct
  Llama-3-70B-Instruct
  其他
```

### 步骤 7: 确认配置

检查配置摘要，确认无误后按回车。

**配置文件位置：**
- Mac/Linux: `~/.openclaw/config.json`
- Windows: `C:\Users\你的用户名\.openclaw\config.json`

## 连接聊天应用

### 以 Telegram 为例

### 步骤 1: 创建 Telegram 机器人

1. 在 Telegram 搜索 `@BotFather`
2. 发送命令 `/newbot`
3. 按提示输入：
   - 机器人名称（如：My Assistant）
   - 机器人用户名（如：my_test_bot，必须以 bot 结尾）

### 步骤 2: 获取 Bot Token

BotFather 会返回类似以下信息：

```
Done! Congratulations on your new bot.
You will find it at t.me/my_test_bot

Use this token to access the HTTP API:
1234567890:ABCdefGHIjklMNOpqrsTUVwxyz

Keep your token secure!
```

**复制 Token**（上面那串字符），这是连接的关键。

### 步骤 3: 在 OpenClaw 中配置

```bash
openclaw connect telegram
```

粘贴刚才复制的 Bot Token。

### 步骤 4: 验证连接

命令执行成功后，会显示：

```
✅ Telegram 连接成功！
机器人地址：t.me/my_test_bot
```

### 其他聊天应用

**WhatsApp:**
```bash
openclaw connect whatsapp
```
需要扫描二维码连接 WhatsApp 账号。

**Discord:**
```bash
openclaw connect discord
```
需要创建 Discord 应用并获取 Bot Token。

## 测试机器人

### 步骤 1: 打开聊天应用

## 常见问题

### Q1: 安装时提示权限错误

**解决方案**: 使用 sudo

```bash
sudo npm install -g openclaw
```

### Q2: 连接 Telegram 时收不到回复

**检查清单：**
- [ ] Bot Token 是否正确
- [ ] 机器人是否已启动（在 Telegram 中点击"Start"）
- [ ] 网络是否通畅
- [ ] API Key 是否有效

### Q3: AI 响应很慢

**可能原因：**
- 网络延迟
- AI 服务负载高
- 模型太大

**解决方案：**
- 尝试切换模型（如 Qwen2.5-32B-Instruct）
- 检查网络连接
- 避开使用高峰期

### Q4: 配置文件在哪里？

**默认位置：**

| 系统 | 路径 |
|------|------|
| Mac/Linux | `~/.openclaw/config.json` |
| Windows | `C:\Users\用户名\.openclaw\config.json` |

**查看配置：**
```bash
openclaw config get
```

### Q5: 如何更新 OpenClaw？

```bash
npm update -g openclaw
```
