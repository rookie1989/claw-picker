## 🛠️ 准备工作

### 1. 确认已安装 OpenClaw

```bash
openclaw --version
```

### 2. 配置必要技能

确保以下技能已启用：
- `feishu-calendar` - 日历查询
- `weather` - 天气查询
- `himalaya` - 邮件读取（可选）

### 3. 准备配置文件

创建 `~/.openclaw/skills/morning-briefing/SKILL.md`

## 📝 步骤 1：创建技能文件

### SKILL.md

```markdown
# morning-briefing

晨间简报技能，每日自动汇总邮件、日历、天气、新闻。

**当以下情况时使用此技能**：
(1) 用户要求"晨间简报"、"今日简报"、"早上好"
(2) 定时任务触发（每天 8:00）
```

### skill.js

```javascript
const { exec } = require('child_process');
const { promisify } = require('util');
const execAsync = promisify(exec);

module.exports = {
  name: 'morning-briefing',
  execute: async (context) => {
    const date = new Date().toLocaleDateString('zh-CN', {
      year: 'numeric',
      month: '2-digit',
      day: '2-digit'
    });

    // 1. 获取日历
    const calendar = await getCalendar();
    
    // 2. 获取天气
    const weather = await getWeather();
    
    // 3. 获取邮件（可选）
    const emails = await getEmails();
    
    // 4. 获取新闻（可选）
    const news = await getNews();
    
    // 5. 组装简报
    const briefing = `
☀️ 早安！今日简报 ${date}

📅 今日日程
${calendar}

🌤️ 天气
${weather}

📬 邮件摘要
${emails}

📰 热点新闻
${news}

💡 提醒
- 今天提交周报
- 明天是截止日期
    `.trim();

    return briefing;
  }
};

async function getCalendar() {
  // 调用飞书日历 API
  // 返回今日日程列表
  return '- 09:30 团队站会\n- 14:00 产品评审会';
}

async function getWeather() {
  // 调用天气 API
  const { stdout } = await execAsync('curl -s "wttr.in/Shanghai?format=%C+%t"');
  return `上海 今天 ${stdout.trim()}`;
}

async function getEmails() {
  // 调用邮件 API
  return '- 3 封未读邮件（1 封紧急）';
}

async function getNews() {
  // 调用新闻 API
  return '- OpenClaw 发布新版本 v1.2.0';
}
```

## 📝 步骤 2：配置定时任务

### 方法 1：使用 Cron 插件

在 `~/.openclaw/config.json` 中添加：

```json
{
  "cron": {
    "jobs": [
      {
        "name": "晨间简报",
        "schedule": {
          "kind": "cron",
          "expr": "0 8 * * *"
        },
        "payload": {
          "kind": "agentTurn",
          "message": "生成今日晨间简报"
        },
        "enabled": true
      }
    ]
  }
}
```

### 方法 2：使用系统 Cron

```bash
# 编辑 crontab
crontab -e

# 添加任务（每天 8:00）
0 8 * * * openclaw send "生成今日晨间简报"
```

### 常见问题

**Q: 技能未触发？**  
A: 检查 cron 配置是否正确，网关是否重启。

**Q: 数据源返回空？**  
A: 检查 API 凭证和网络连接。

**Q: 格式混乱？**  
A: 确保 Markdown 格式正确，避免特殊字符。
