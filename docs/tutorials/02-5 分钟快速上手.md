## 步骤 1: 安装 OpenClaw (1 分钟)

打开终端，运行：

```bash
npm install -g openclaw
```

验证安装：

```bash
openclaw --version
```

看到版本号即成功！

---

## 步骤 2: 初始化配置 (2 分钟)

### 运行向导

```bash
openclaw onboard
```

### 快速选择

按以下顺序选择（全程回车即可）：

| 问题 | 推荐选择 |
|------|----------|
| 运行方式 | 本地运行 |
| 聊天应用 | Telegram |
| AI 服务 | 硅基流动 |
| API Key | 粘贴你的 Key |
| 模型 | Qwen2.5-72B-Instruct |

> 💡 **没有 API Key?** 立即前往 [siliconflow.cn](https://siliconflow.cn) 注册获取（1 分钟）

### 配置完成

向导会显示：
```
✅ 配置已生成
配置文件：~/.openclaw/config.json
```

---

## 步骤 3: 连接 Telegram (3 分钟)

### 3.1 创建机器人

1. 打开 Telegram，搜索 `@BotFather`
2. 发送 `/newbot`
3. 输入机器人名称（任意，如：My Assistant）
4. 输入用户名（必须以 bot 结尾，如：my_test_bot）

### 3.2 获取 Token

BotFather 会返回：
```
Use this token: 1234567890:ABCdefGHIjklMNOpqrsTUVwxyz
```

**复制这个 Token！**

### 3.3 配置 OpenClaw

终端运行：

```bash
openclaw connect telegram
```

粘贴刚才复制的 Token。

### 3.4 启动机器人

1. 在 Telegram 中找到你的机器人
2. 点击 **Start** 按钮
3. 机器人会显示"已连接"

---

## 步骤 4: 测试机器人 (1 分钟)

### 发送测试消息

在 Telegram 中给你的机器人发送：

```
你好
```

### 预期回复

你应该在几秒内收到 AI 的友好回复。

## 🚀 快速参考卡

### 常用命令

```bash
# 查看状态
openclaw status

# 查看配置
openclaw config get

# 重新配置
openclaw onboard

# 连接其他应用
openclaw connect whatsapp
openclaw connect discord
```

### 配置文件位置

| 系统 | 路径 |
|------|------|
| Mac/Linux | `~/.openclaw/config.json` |
| Windows | `C:\Users\用户名\.openclaw\config.json` |

### 快速故障排查

| 问题 | 解决方案 |
|------|----------|
| 收不到回复 | 检查机器人是否启动（点击 Start） |
| 提示权限错误 | 命令前加 `sudo` |
| API Key 无效 | 重新生成 Key 并更新配置 |
| 网络超时 | 检查网络连接，切换 AI 服务 |
