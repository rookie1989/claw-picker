# ❓ 配置问题 FAQ

OpenClaw 配置文件相关的常见问题解答。

## 📄 配置文件基础

### Q: 配置文件在哪？

**A:** 
```
~/.openclaw/config.json
```

查看配置：
```bash
cat ~/.openclaw/config.json
```

### Q: 配置文件格式？

**A:** 标准 JSON 格式：
```json
{
  "server": {
    "port": 3000
  },
  "plugins": {
    "entries": ["telegram"]
  }
}
```

**注意**：
- 使用双引号 `"`
- 最后一个元素后不能有逗号
- 注释不支持（JSON 标准）

### Q: 如何编辑配置？

**A:** 
```bash
# 方法 1：使用编辑器
nano ~/.openclaw/config.json

# 方法 2：使用命令
openclaw config edit

# 方法 3：使用 patch
openclaw config patch --path server.port --value 3001
```

## 🔧 配置验证

### Q: 如何验证配置？

**A:** 
```bash
openclaw config validate
```

输出示例：
```
✅ 配置验证通过
```

或：
```
❌ 配置错误：plugins.entries 不能为空
```

### Q: 配置错误如何修复？

**A:** 
1. 查看错误信息
2. 使用 JSON 验证器检查格式
3. 参考配置模板

**在线验证器**：
- https://jsonlint.com/
- https://codebeautify.org/jsonvalidator

## 📦 插件配置

### Q: 如何启用插件？

**A:** 
```json
{
  "plugins": {
    "entries": ["telegram", "whatsapp", "discord"]
  }
}
```

### Q: 如何禁用插件？

**A:** 
```json
{
  "plugins": {
    "entries": ["telegram"]  // 只保留需要的
  }
}
```

### Q: 多插件如何配置？

**A:** 
```json
{
  "plugins": {
    "entries": ["telegram", "discord"],
    "telegram": {
      "token": "BOT_TOKEN_1"
    },
    "discord": {
      "token": "BOT_TOKEN_2"
    }
  }
}
```

## 🤖 模型配置

### Q: 如何切换 AI 模型？

**A:** 
```json
{
  "model": {
    "default": "qwen3.5-plus"
  }
}
```

**常用模型**：
- `qwen3.5-plus` - 通义千问（推荐）
- `qwen3.5-lite` - 快速模型
- `gpt-4` - OpenAI GPT-4
- `claude-3-sonnet` - Anthropic Claude

### Q: 如何配置多个模型提供商？

**A:** 
```json
{
  "model": {
    "default": "gpt-4",
    "providers": {
      "openai": {
        "apiKey": "sk-xxx"
      },
      "bailian": {
        "apiKey": "xxx"
      }
    }
  }
}
```

### Q: 模型响应慢怎么办？

**A:** 
```json
{
  "model": {
    "default": "qwen3.5-lite",  // 使用更快的模型
    "timeout": 60000            // 增加超时时间
  }
}
```

## 🛠️ 技能配置

### Q: 如何启用技能？

**A:** 
```json
{
  "skills": {
    "entries": ["weather", "github", "feishu-bitable"]
  }
}
```

### Q: 自定义技能如何配置？

**A:** 
```json
{
  "skills": {
    "entries": ["my-custom-skill"],
    "paths": ["~/.openclaw/skills"]
  }
}
```

### Q: 技能不生效怎么办？

**A:** 
1. 检查技能是否在 entries 中
2. 验证 SKILL.md 格式
3. 重启网关
4. 查看日志

## ⏰ 定时任务配置

### Q: 如何添加定时任务？

**A:** 
```json
{
  "cron": {
    "jobs": [
      {
        "name": "晨间简报",
        "schedule": {
          "kind": "cron",
          "expr": "0 8 * * *"
        },
        "payload": {
          "kind": "agentTurn",
          "message": "生成晨间简报"
        }
      }
    ]
  }
}
```

### Q: Cron 表达式怎么写？

**A:** 
```
分 时 日 月 周

0 8 * * *    → 每天 8:00
*/30 * * * * → 每 30 分钟
0 10 * * 1   → 每周一 10:00
```

**在线生成器**：https://crontab.guru/

## 🔐 安全配置

### Q: 如何限制访问用户？

**A:** 
```json
{
  "security": {
    "allowlist": ["+86138****", "@trusted_user"]
  }
}
```

### Q: 如何设置速率限制？

**A:** 
```json
{
  "security": {
    "rateLimit": {
      "requests": 100,
      "window": 3600
    }
  }
}
```

### Q: 如何配置 API 密钥？

**A:** 
```json
{
  "env": {
    "GITHUB_TOKEN": "ghp_xxx",
    "OPENWEATHER_API_KEY": "xxx"
  }
}
```

**或使用环境变量**：
```bash
export GITHUB_TOKEN=ghp_xxx
```

## 🌐 服务器配置

### Q: 如何更改监听端口？

**A:** 
```json
{
  "server": {
    "port": 3001
  }
}
```

### Q: 如何启用 HTTPS？

**A:** 
```json
{
  "server": {
    "https": true,
    "cert": "/path/to/cert.pem",

  }
}
```

### Q: 如何配置 CORS？

**A:** 
```json
{
  "server": {
    "cors": true,
    "corsOrigins": ["https://example.com"]
  }
}
```

## 🐛 配置调试

### Q: 如何查看当前配置？

**A:** 
```bash
openclaw config get
```

### Q: 如何比较配置差异？

**A:** 
```bash
# 导出配置
openclaw config get > config1.json

# 修改后再次导出
openclaw config get > config2.json

# 比较差异
diff config1.json config2.json
```

### Q: 配置不生效怎么办？

**A:** 
1. 验证配置格式
2. 重启网关
3. 查看日志确认加载
4. 检查配置文件路径

## 📚 配置模板

### 最小配置

```json
{
  "plugins": {
    "entries": []
  }
}
```

### 完整配置示例

```json
{
  "server": {
    "host": "0.0.0.0",
    "port": 3000
  },
  "plugins": {
    "entries": ["telegram", "feishu"],
    "telegram": {"token": "xxx"},
    "feishu": {"appId": "cli_xxx", "userToken": "u-xxx"}
  },
  "model": {
    "default": "qwen3.5-plus"
  },
  "skills": {
    "entries": ["weather", "github"]
  },
  "cron": {
    "jobs": [
      {
        "name": "晨间简报",
        "schedule": {"kind": "cron", "expr": "0 8 * * *"},
        "payload": {"kind": "agentTurn", "message": "晨间简报"}
      }
    ]
  },
  "security": {
    "allowlist": ["@admin"],
    "rateLimit": {"requests": 100, "window": 3600}
  },
  "session": {
    "maxMessages": 50,
    "summarize": true
  }
}
```
