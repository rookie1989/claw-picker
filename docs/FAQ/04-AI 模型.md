# ❓ AI 模型 FAQ

AI 模型配置、切换、优化相关问题解答。

## 🤖 模型选择

### Q: 支持哪些 AI 模型？

**A:** OpenClaw 支持多种模型提供商：

| 提供商 | 模型 | 说明 |
|--------|------|------|
| 通义千问 | qwen3.5-plus | 推荐，中文优化 |
| 通义千问 | qwen3.5-lite | 快速，低成本 |
| OpenAI | gpt-4 | 最强，英文优化 |
| OpenAI | gpt-3.5-turbo | 快速，经济 |
| Anthropic | claude-3-sonnet | 平衡性能和成本 |
| Anthropic | claude-3-opus | 最强推理 |

### Q: 如何选择适合的模型？

**A:** 

**个人使用**：
- 推荐：`qwen3.5-plus`
- 理由：中文好，性价比高

**生产环境**：
- 推荐：`qwen3.5-plus` + `qwen3.5-lite`（降级）
- 理由：有备份，稳定性高

**英文场景**：
- 推荐：`gpt-4` 或 `claude-3-sonnet`
- 理由：英文理解更好

**成本敏感**：
- 推荐：`qwen3.5-lite`
- 理由：速度快，成本低

### Q: 如何切换模型？

**A:** 
```json
{
  "model": {
    "default": "qwen3.5-plus"
  }
}
```

**临时切换**：
```
/switch_model gpt-4
```

### Q: 如何配置降级模型？

**A:** 
```json
{
  "model": {
    "default": "qwen3.5-plus",
    "fallback": "qwen3.5-lite"
  }
}
```

当主模型失败时，自动使用降级模型。

## 🔑 API 密钥配置

### Q: 通义千问密钥如何获取？

**A:** 
1. 访问 [阿里云百炼](https://bailian.console.aliyun.com/)
2. 登录阿里云账号
3. 开通百炼服务
4. 创建 API Key
5. 复制 Key 到配置

**配置**：
```json
{
  "model": {
    "providers": {
      "bailian": {
        "apiKey": "sk-xxx"
      }
    }
  }
}
```

### Q: OpenAI API Key 如何获取？

**A:** 
1. 访问 [OpenAI Platform](https://platform.openai.com/)
2. 登录/注册账号
3. 左侧菜单 → API Keys
4. 点击 "Create new secret key"
5. 复制 Key（只显示一次）

**配置**：
```json
{
  "model": {
    "providers": {
      "openai": {
        "apiKey": "sk-xxx",
        "baseUrl": "https://api.openai.com/v1"
      }
    }
  }
}
```

### Q: 如何安全存储 API 密钥？

**A:** 

**方法 1：环境变量**
```bash
export OPENAI_API_KEY=sk-xxx
export BAILIAN_API_KEY=sk-xxx
```

**方法 2：使用 1Password**
```json
{
  "plugins": {
    "1password": {
      "vault": "Personal",
      "item": "API Keys"
    }
  }
}
```

**方法 3：配置文件（注意权限）**
```bash
chmod 600 ~/.openclaw/config.json
```

## ⚡ 性能优化

### Q: 模型响应慢怎么办？

**A:** 

1. **切换到更快的模型**：
```json
{
  "model": {
    "default": "qwen3.5-lite"
  }
}
```

2. **增加超时时间**：
```json
{
  "model": {
    "timeout": 60000
  }
}
```

3. **检查网络**：
```bash
curl -I https://api.openai.com
```

4. **使用国内节点**：
```json
{
  "model": {
    "providers": {
      "bailian": {
        "endpoint": "dashscope.aliyuncs.com"
      }
    }
  }
}
```

### Q: 如何减少 Token 消耗？

**A:** 

1. **限制历史消息**：
```json
{
  "session": {
    "maxMessages": 30
  }
}
```

2. **启用摘要**：
```json
{
  "session": {
    "summarize": true
  }
}
```

3. **优化提示词**：
```json
{
  "systemPrompt": "简洁回答，不超过 100 字"
}
```

4. **使用更小的模型**：
```json
{
  "model": {
    "default": "qwen3.5-lite"
  }
}
```

### Q: 如何提高回答质量？

**A:** 

1. **使用更强的模型**：
```json
{
  "model": {
    "default": "qwen3.5-plus"
  }
}
```

2. **优化系统提示词**：
```json
{
  "systemPrompt": "你是专业的助手，回答要准确、详细、有条理"
}
```

3. **提供更多上下文**：
```json
{
  "session": {
    "maxMessages": 50
  }
}
```

4. **使用 Few-Shot**：
```json
{
  "systemPrompt": "示例：\n用户：你好\n助手：您好！有什么可以帮助您的？"
}
```

## 🔄 多模型配置

### Q: 如何配置多个模型提供商？

**A:** 
```json
{
  "model": {
    "default": "gpt-4",
    "providers": {
      "openai": {
        "apiKey": "sk-xxx",
        "baseUrl": "https://api.openai.com/v1"
      },
      "bailian": {
        "apiKey": "sk-yyy"
      },
      "anthropic": {
        "apiKey": "sk-ant-zzz"
      }
    }
  }
}
```

### Q: 如何按场景使用不同模型？

**A:** 
```json
{
  "model": {
    "routing": {
      "creative": "gpt-4",
      "coding": "qwen3.5-plus",
      "chat": "qwen3.5-lite"
    }
  }
}
```

### Q: 如何实现负载均衡？

**A:** 
```json
{
  "model": {
    "loadBalance": {
      "enabled": true,
      "models": [
        {"name": "qwen3.5-plus", "weight": 70},
        {"name": "gpt-4", "weight": 30}
      ]
    }
  }
}
```

## 🐛 常见问题

### Q: API Key 无效？

**A:** 
1. 检查 Key 是否正确复制
2. 检查 Key 是否过期
3. 检查账户余额
4. 重新生成 Key

### Q: 速率限制错误？

**A:** 
```
Error: 429 Too Many Requests
```

**解决**：
1. 降低请求频率
2. 升级账户套餐
3. 使用多个 Key 轮换

```json
{
  "model": {
    "providers": {
      "openai": {
        "apiKeys": ["sk-1", "sk-2", "sk-3"],
        "keyRotation": true
      }
    }
  }
}
```

### Q: 模型不可用？

**A:** 
```
Error: Model not found
```

**解决**：
1. 检查模型名称拼写
2. 确认模型在该区域可用
3. 切换到备用模型

### Q: 中文回答质量差？

**A:** 
1. 使用通义千问（中文优化）
2. 在提示词中指定语言
3. 提供中文示例

```json
{
  "systemPrompt": "请用中文回答，保持专业"
}
```
