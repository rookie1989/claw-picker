# ❓ 渠道连接 FAQ

Telegram、WhatsApp、Discord、飞书等聊天应用连接问题解答。

## 📱 Telegram

### Q: Bot Token 在哪获取？

**A:** 
1. 打开 Telegram，搜索 [@BotFather](https://t.me/BotFather)
2. 发送 `/newbot`
3. 按提示输入 Bot 名称和用户名
4. 获取 Token（格式：`123456789:ABCdefGHIjklMNOpqrsTUVwxyz`）

### Q: Telegram 无法接收消息？

**A:** 
1. **检查 Token**：
```json
{
  "plugins": {
    "telegram": {
      "token": "正确的 BOT_TOKEN"
    }
  }
}
```

2. **检查网关**：
```bash
openclaw gateway status
```

3. **查看日志**：
```bash
openclaw logs --grep telegram
```

### Q: Telegram  webhook 还是 polling？

**A:** 
- **Polling**（推荐新手）：简单，无需公网 IP
- **Webhook**：实时，需要公网 IP 和 HTTPS

**Polling 配置**：
```json
{
  "telegram": {
    "token": "xxx",
    "webhook": false
  }
}
```

**Webhook 配置**：
```json
{
  "telegram": {
    "token": "xxx",
    "webhook": true,
    "webhookUrl": "https://your-domain.com/telegram"
  }
}
```

### Q: 如何邀请 Bot 进群？

**A:** 
1. 在 Telegram 搜索 Bot 用户名
2. 点击进入 Bot 聊天
3. 点击 Bot 名称 → 添加到群组
4. 选择目标群组
5. 设置管理员权限（可选）

## 💬 WhatsApp

### Q: WhatsApp Business API 如何申请？

**A:** 
**方法 1：Meta 官方**
1. 访问 [Meta for Developers](https://developers.facebook.com/)
2. 创建应用 → 选择 Business
3. 添加 WhatsApp 产品
4. 提交商业验证

**方法 2：第三方服务商**
- Twilio
- MessageBird
- 360dialog

### Q: Phone Number ID 在哪？

**A:** 
1. 访问 [Meta Business Suite](https://business.facebook.com/)
2. 选择 WhatsApp 账号
3. 设置 → 账号信息
4. 找到 Phone Number ID

### Q: Webhook 配置失败？

**A:** 
1. **验证 Token**：
```json
{
  "whatsapp": {
    "verifyToken": "自定义验证令牌"
  }
}
```

2. **配置 Webhook URL**：
```
https://your-domain.com/whatsapp
```

3. **订阅事件**：
- messages
- message_deliveries
- message_reads

### Q: 消息发送失败？

**A:** 
检查模板消息审核：
- 营销消息需要预审
- 服务消息可直接发送
- 24 小时会话窗口

## 🎮 Discord

### Q: Discord Bot Token 在哪？

**A:** 
1. 访问 [Discord Developer Portal](https://discord.com/developers/applications)
2. 创建应用
3. 左侧菜单 → Bot
4. 点击 "Reset Token" 获取

### Q: Bot 无法加入服务器？

**A:** 
1. **生成邀请链接**：
```
https://discord.com/api/oauth2/authorize?client_id=CLIENT_ID&permissions=8&scope=bot
```

2. **检查权限**：
- 读取消息
- 发送消息
- 读取消息历史

3. **检查服务器设置**：
- 管理员是否禁止 Bot 加入

### Q: 消息内容为空？

**A:** 启用 Privileged Gateway Intents：
1. Developer Portal → Bot
2. 向下滚动到 "Privileged Gateway Intents"
3. 启用 "MESSAGE CONTENT INTENT"
4. 保存

### Q: 如何配置 Intents？

**A:** 
```json
{
  "discord": {
    "token": "xxx",
    "intents": [
      "GUILDS",
      "GUILD_MESSAGES",
      "MESSAGE_CONTENT"
    ]
  }
}
```

## 📧 飞书

### Q: 飞书应用如何创建？

**A:** 
1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 登录企业账号
3. 点击"创建企业自建应用"
4. 填写应用信息

### Q: 权限申请失败？

**A:** 
1. **检查账号权限**：需要管理员
2. **等待审批**：部分权限需审批
3. **检查配额**：部分权限有数量限制

### Q: 事件订阅不生效？

**A:** 
1. **验证 URL**：
```
https://your-domain.com/feishu/event
```

2. **配置验证 Token**：
```json
{
  "feishu": {
    "verificationToken": "xxx"
  }
}
```

3. **订阅事件**：
- 接收消息
- 消息已读
- 应用版本更新

### Q: User Token 如何获取？

**A:** 
**方法 1：OAuth 授权**
1. 配置重定向 URL
2. 用户访问授权页面
3. 获取授权码
4. 换取 User Token

**方法 2：手动获取**
1. 开放平台 → 应用 → 凭证
2. 点击"获取用户身份凭证"
3. 复制 Token

## 🔗 通用问题

### Q: 多通道如何配置？

**A:** 
```json
{
  "plugins": {
    "entries": ["telegram", "discord", "feishu"],
    "telegram": {"token": "xxx"},
    "discord": {"token": "xxx"},
    "feishu": {"appId": "cli_xxx", "userToken": "u-xxx"}
  }
}
```

### Q: 如何区分不同渠道？

**A:** 在技能中检查 `context.channel`：
```javascript
if (context.channel === 'telegram') {
  // Telegram 特定逻辑
} else if (context.channel === 'feishu') {
  // 飞书特定逻辑
}
```

### Q: 消息格式不统一？

**A:** 使用平台适配层：
```javascript
function formatMessage(platform, content) {
  switch (platform) {
    case 'telegram':
      return {parse_mode: 'Markdown', text: content};
    case 'discord':
      return {content: content};
    case 'feishu':
      return {msg_type: 'text', content: {text: content}};
  }
}
```

### Q: 连接超时怎么办？

**A:** 
```json
{
  "plugins": {
    "telegram": {
      "polling": {
        "timeout": 60,
        "interval": 2000
      }
    }
  }
}
```

### Q: 如何测试连接？

**A:** 
```bash
# Telegram
curl "https://api.telegram.org/botTOKEN/getMe"

# Discord
curl -H "Authorization: Bot TOKEN" "https://discord.com/api/users/@me"

# 飞书
curl -H "Authorization: Bearer TOKEN" "https://open.feishu.cn/open-apis/auth/v3/user_info"
```

## 🐛 调试技巧

### 查看连接日志

```bash
# 实时日志
openclaw logs --follow --grep telegram

# 错误日志
openclaw logs --level error --grep discord
```

### 测试 API 连通性

```bash
# Telegram
curl -s "https://api.telegram.org/botTOKEN/getUpdates" | jq

# 飞书
curl -s -H "Authorization: Bearer TOKEN" "https://open.feishu.cn/open-apis/auth/v3/user_info" | jq
```

### 重启插件

```bash
# 重启网关
openclaw gateway restart

# 或热重载（如果支持）
openclaw plugins reload
```
