# ❓ 工具使用 FAQ

OpenClaw 技能、命令、功能使用相关问题解答。

## 🛠️ 技能使用

### Q: 如何启用技能？

**A:** 
```json
{
  "skills": {
    "entries": ["weather", "github", "feishu-bitable"]
  }
}
```

**重启网关**：
```bash
openclaw gateway restart
```

### Q: 技能不响应怎么办？

**A:** 

1. **检查是否启用**：
```bash
openclaw skills list
```

2. **查看技能日志**：
```bash
openclaw logs --grep skill
```

3. **验证 SKILL.md 格式**：
```markdown
# skill-name

描述...

**当以下情况时使用此技能**：
(1) 触发条件 1
(2) 触发条件 2
```

4. **重启网关**：
```bash
openclaw gateway restart
```

### Q: 如何调用技能？

**A:** 

**自动触发**（根据消息内容）：
```
北京天气怎么样？
→ 自动调用 weather 技能
```

**手动调用**（如果支持）：
```
/run weather Beijing
```

### Q: 自定义技能如何开发？

**A:** 

1. **创建目录**：
```bash
mkdir -p ~/.openclaw/skills/my-skill
```

2. **编写 SKILL.md**：
```markdown
# my-skill

技能描述

**当以下情况时使用此技能**：
(1) 条件 1
```

3. **编写 skill.js**（可选）：
```javascript
module.exports = {
  name: 'my-skill',
  execute: async (context) => {
    return '结果';
  }
};
```

4. **启用技能**：
```json
{
  "skills": {
    "entries": ["my-skill"]
  }
}
```

详见：[技能开发教程](../advanced/skill-development.md)

## 📋 命令使用

### Q: 如何查看可用命令？

**A:** 
```bash
openclaw --help
openclaw gateway --help
```

**会话内命令**：
```
/help
```

### Q: 如何创建新会话？

**A:** 
```
/new
```

这会清空上下文，开始新对话。

### Q: 如何重置当前会话？

**A:** 
```
/reset
```

效果同 `/new`。

### Q: 如何查看会话状态？

**A:** 
```
/status
```

显示：
- 当前模型
- 使用量
- 会话时长
- 消息数量

### Q: 如何查看历史会话？

**A:** 
```
/sessions
```

列出所有会话，可选择恢复。

## 📅 定时任务

### Q: 如何添加定时任务？

**A:** 
```bash
/cron add "每天早上 8 点生成晨间简报"
```

**或配置文件**：
```json
{
  "cron": {
    "jobs": [
      {
        "name": "晨间简报",
        "schedule": {"kind": "cron", "expr": "0 8 * * *"},
        "payload": {"kind": "agentTurn", "message": "生成晨间简报"}
      }
    ]
  }
}
```

### Q: 如何查看定时任务？

**A:** 
```bash
/cron list
```

### Q: 如何删除定时任务？

**A:** 
```bash
/cron remove <job-id>
```

### Q: 如何手动触发任务？

**A:** 
```bash
/cron run <job-id>
```

## 🔌 插件管理

### Q: 如何查看已安装插件？

**A:** 
```bash
openclaw plugins list
```

### Q: 如何安装插件？

**A:** 
```bash
npm install -g @openclaw/plugin-name
```

### Q: 如何禁用插件？

**A:** 
```json
{
  "plugins": {
    "entries": ["telegram"]  // 只保留需要的
  }
}
```

### Q: 如何卸载插件？

**A:** 
```bash
npm uninstall -g @openclaw/plugin-name
```

## 💬 消息功能

### Q: 如何发送消息？

**A:** 
在聊天应用中直接发消息给 Bot 即可。

### Q: 如何引用消息？

**A:** 
不同平台方式不同：

**Telegram**：回复消息
**Discord**：回复线程
**飞书**：引用消息

### Q: 如何发送文件？

**A:** 
直接上传文件到聊天，Bot 会自动处理。

### Q: 支持哪些文件类型？

**A:** 
- ✅ 图片（jpg, png, gif, webp）
- ✅ 文档（pdf, doc, docx, txt）
- ✅ 代码文件
- ✅ 压缩包

## 🎯 高级功能

### Q: 如何使用子 Agent？

**A:** 
```bash
/spawn 帮我写一个 Python 脚本
```

**查看子 Agent**：
```bash
/subagents
```

**终止子 Agent**：
```bash
/kill <agent-id>
```

### Q: 如何使用 ACP 编码会话？

**A:** 
```bash
/acp 帮我修复这个 bug
```

这会启动一个独立的编码会话。

### Q: 如何等待子 Agent 完成？

**A:** 
```bash
/yield
```

### Q: 如何切换到不同模型？

**A:** 
```bash
/switch_model gpt-4
```

**或配置**：
```json
{
  "model": {
    "default": "gpt-4"
  }
}
```

## 📊 统计与监控

### Q: 如何查看使用量？

**A:** 
```
/status
```

显示：
- Token 使用
- 会话时长
- 消息数量
- 成本估算

### Q: 如何查看日志？

**A:** 
```bash
# 实时日志
openclaw logs --follow

# 最近 100 行
openclaw logs --tail 100

# 错误日志
openclaw logs --level error

# 搜索关键词
openclaw logs --grep "Error"
```

### Q: 如何导出日志？

**A:** 
```bash
openclaw logs --output /tmp/openclaw.log
```

## 📱 移动端功能

### Q: 如何配对移动设备？

**A:** 
1. 服务器运行 `openclaw node pairing`
2. 手机 App 扫描二维码
3. 完成配对

### Q: 移动端能做什么？

**A:** 
- 查看摄像头
- 屏幕录制
- 位置获取
- 通知管理
- 运行命令

### Q: 如何查看已配对设备？

**A:** 
```bash
/nodes status
```

## 🐛 故障排查

### Q: 命令无响应？

**A:** 
1. 检查网关状态
2. 查看日志
3. 重启网关

### Q: 功能突然失效？

**A:** 
1. 检查配置是否被修改
2. 查看错误日志
3. 回滚到上一个可用版本

### Q: 如何重置所有配置？

**A:** 
```bash
# 备份
mv ~/.openclaw ~/.openclaw.backup

# 重新初始化
openclaw onboard
```
