# FAQ - 工具使用

> 浏览器、搜索、文件操作等工具使用问题解答

---

## 浏览器工具

### Q1: 浏览器无法启动

**原因：** 缺少浏览器或依赖

**解决方案：**

**Mac:**
```bash
# 安装 Chrome
brew install --cask google-chrome
```

**Linux:**
```bash
# Ubuntu/Debian
wget https://dl.google.com/linux/direct/google-chrome-stable_current_amd64.deb
sudo dpkg -i google-chrome-stable_current_amd64.deb
```

**Windows:**
下载 Chrome: https://www.google.com/chrome/

**配置：**
```yaml
skills:
  - name: browser
    enabled: true
    config:
      browser: chrome  # 或 firefox, safari
      headless: true
```

---

### Q2: 浏览器访问被拒绝

**原因：** 网站反爬虫或需要登录

**解决方案：**

1. **配置 User-Agent：**
   ```yaml
   skills:
     - name: browser
       config:
         userAgent: "Mozilla/5.0 (Macintosh; Intel Mac OS X 10_15_7)..."
   ```

2. **使用有头模式调试：**
   ```yaml
   skills:
     - name: browser
       config:
         headless: false  # 显示浏览器窗口
   ```

3. **配置 Cookie/登录状态**

---

### Q3: 页面加载超时

**解决方案：**

```yaml
skills:
  - name: browser
    config:
      timeout: 60000  # 增加超时时间（毫秒）
      waitForLoad: true
      waitUntil: networkidle  # 等待网络空闲
```

---

### Q4: 如何截图？

**命令：**
```bash
openclaw browser screenshot https://example.com --output screenshot.png
```

**技能调用：**
```yaml
skills:
  - name: browser
    config:
      screenshot:
        enabled: true
        fullPage: true  # 完整页面
```

## 搜索工具

### Q5: 网络搜索不工作

**原因：** 未配置搜索 API

**解决方案：**

**使用 Brave Search（推荐）：**
```yaml
skills:
  - name: search
    enabled: true
    config:
      engine: brave
      apiKey: YOUR_BRAVE_API_KEY  # 获取：https://brave.com/search/api/
```

**使用 Google Custom Search：**
```yaml
skills:
  - name: search
    config:
      engine: google
      apiKey: YOUR_GOOGLE_API_KEY
      cseId: YOUR_CSE_ID
```

---

### Q6: 搜索结果不准确

**优化：**

```yaml
skills:
  - name: search
    config:
      engine: brave
      count: 10          # 结果数量
      freshness: week    # 时间过滤：day/week/month
      language: zh       # 语言
      country: CN        # 地区
```

---

### Q7: 搜索频率受限

**原因：** API 调用超限

**解决方案：**

1. **降低频率：**
   ```yaml
   skills:
     - name: search
       config:
         rateLimit:
           requestsPerMinute: 5
   ```

2. **启用缓存：**
   ```yaml
   skills:
     - name: search
       config:
         cache:
           enabled: true
           ttl: 3600  # 1 小时
   ```

3. **升级 API 配额**

## 文件操作

### Q8: 文件读写权限错误

**原因：** 目录权限不足

**解决方案：**

**Mac/Linux:**
```bash
# 检查权限
ls -la ~/Documents

# 修改权限
chmod 755 ~/Documents
```

**配置允许目录：**
```yaml
skills:
  - name: files
    enabled: true
    config:
      allowedDirs:
        - ~/Documents
        - /tmp
        - ~/.openclaw/data
```

---

### Q9: 无法访问某些目录

**原因：** 安全限制

**解决方案：**

```yaml
skills:
  - name: files
    config:
      allowedDirs:
        - /home/user/work
        - /shared/project
      denyDirs:
        - /etc
        - /var/log
      maxFileSize: 10MB  # 最大文件大小
```

---

### Q10: 大文件处理失败

**原因：** 内存限制

**解决方案：**

```yaml
skills:
  - name: files
    config:
      maxFileSize: 50MB      # 增加限制
      streamLargeFiles: true  # 流式处理
      chunkSize: 1MB         # 分块大小
```

## 日历工具

### Q11: 日历无法同步

**原因：** OAuth 认证问题

**解决方案：**

**Google Calendar:**
```bash
# 重新授权
openclaw calendar auth google
```

**配置：**
```yaml
skills:
  - name: calendar
    config:
      provider: google
      clientId: YOUR_CLIENT_ID
      clientSecret: YOUR_CLIENT_SECRET
```

---

### Q12: 如何创建日程？

**命令：**
```bash
openclaw calendar create "团队会议" --time "2026-03-20T14:00:00+08:00" --duration 60
```

**自然语言：**
```
"明天下午 2 点创建一个 1 小时的团队会议"
```

## 任务工具

### Q13: 如何管理待办事项？

**命令：**
```bash
# 创建任务
openclaw task create "完成报告" --due "2026-03-20"

# 查看任务
openclaw task list

# 完成任务
openclaw task complete 123  # 任务 ID
```

---

### Q14: 任务提醒不工作

**排查：**

1. 检查提醒技能是否启用：
   ```yaml
   skills:
     - name: reminder
       enabled: true
   ```

2. 检查 Gateway 是否运行
3. 查看提醒数据：
   ```bash
   cat ~/.openclaw/data/reminders.json
   ```

## 自定义技能

### Q15: 如何开发自定义技能？

**步骤：**

1. **创建技能文件：**
   ```javascript
   // skills/my-skill.js
   async function mySkill(params) {
     return { result: 'Hello from custom skill!' };
   }
   
   module.exports = { mySkill };
   ```

2. **配置技能：**
   ```yaml
   skills:
     - name: my-skill
       script: ./skills/my-skill.js
       triggers:
         - keywords: ["我的技能"]
   ```

3. **测试技能：**
   ```bash
   openclaw skill test my-skill
   ```

---

### Q16: 技能加载失败

**排查：**

1. 检查脚本路径是否正确
2. 验证 JavaScript 语法：
   ```bash
   node --check ./skills/my-skill.js
   ```
3. 查看 Gateway 日志：
   ```bash
   cat ~/.openclaw/logs/gateway.log | grep skill
   ```
4. 确保依赖已安装：
   ```bash
   npm install  # 在技能目录
   ```

## 性能优化

### Q17: 工具调用慢

**优化：**

```yaml
skills:
  - name: browser
    config:
      pool:
        enabled: true
        size: 3  # 浏览器实例池
      
  - name: search
    config:
      cache:
        enabled: true
        ttl: 1800  # 30 分钟缓存
```

---

### Q18: 内存占用高

**优化：**

```yaml
gateway:
  maxMemory: 2GB  # 限制最大内存
  
skills:
  - name: browser
    config:
      cleanupInterval: 300000  # 5 分钟清理
```
