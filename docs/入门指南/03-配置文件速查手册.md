## 基础配置

### 服务器设置

```json
{
  "server": {
    "host": "0.0.0.0",
    "port": 3000,
    "https": false,
    "cors": true
  }
}
```

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `host` | string | `"0.0.0.0"` | 监听地址 |
| `port` | number | `3000` | 监听端口 |
| `https` | boolean | `false` | 启用 HTTPS |
| `cors` | boolean | `true` | 允许跨域 |

### 日志配置

```json
{
  "logging": {
    "level": "info",
    "format": "json",
    "output": "file",
    "maxFiles": 7
  }
}
```

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `level` | string | `"info"` | 日志级别 |
| `format` | string | `"json"` | 日志格式 |
| `output` | string | `"file"` | 输出方式 |
| `maxFiles` | number | `7` | 保留天数 |

**日志级别**：`debug` < `info` < `warn` < `error`

## 插件配置

### Telegram

```json
{
  "plugins": {
    "telegram": {
      "token": "BOT_TOKEN_HERE",
      "webhook": false,
      "polling": {
        "interval": 1000,
        "timeout": 30
      }
    }
  }
}
```

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `token` | string | ✅ | Bot Token |
| `webhook` | boolean | ❌ | Webhook 模式 |
| `polling.interval` | number | ❌ | 轮询间隔 (ms) |
| `polling.timeout` | number | ❌ | 超时时间 (s) |

### WhatsApp

```json
{
  "plugins": {
    "whatsapp": {
      "phoneNumberId": "123456789",
      "token": "EAAB...",
      "verifyToken": "my_verify_token",
      "webhookPath": "/whatsapp"
    }
  }
}
```

### Discord

```json
{
  "plugins": {
    "discord": {
      "token": "BOT_TOKEN",
      "intents": ["GUILDS", "GUILD_MESSAGES", "MESSAGE_CONTENT"]
    }
  }
}
```

### 飞书

```json
{
  "plugins": {
    "feishu": {
      "appId": "cli_xxx",
      "appSecret": "xxx",
      "userToken": "u-xxx",
      "encryptKey": "xxx",
      "verificationToken": "xxx"
    }
  }
}
```

| 字段 | 说明 |
|------|------|
| `appId` | 应用 ID |
| `appSecret` | 应用密钥 |
| `userToken` | 用户访问令牌 |
| `encryptKey` | 事件订阅加密密钥 |
| `verificationToken` | 事件订阅验证令牌 |

### 多插件配置

```json
{
  "plugins": {
    "entries": ["telegram", "whatsapp", "discord"],
    "telegram": { /* ... */ },
    "whatsapp": { /* ... */ },
    "discord": { /* ... */ }
  }
}
```

## 模型配置

### 基础配置

```json
{
  "model": {
    "default": "qwen3.5-plus",
    "fallback": "qwen3.5-lite",
    "timeout": 30000
  }
}
```

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `default` | string | - | 默认模型 |
| `fallback` | string | - | 降级模型 |
| `timeout` | number | `30000` | 超时 (ms) |

### 多提供商

```json
{
  "model": {
    "default": "gpt-4",
    "providers": {
      "openai": {
        "apiKey": "sk-xxx",
        "baseUrl": "https://api.openai.com/v1"
      },
      "anthropic": {
        "apiKey": "sk-ant-xxx"
      },
      "bailian": {
        "apiKey": "xxx",
        "endpoint": "dashscope.aliyuncs.com"
      }
    }
  }
}
```

### 常用模型

| 提供商 | 模型 | 说明 |
|--------|------|------|
| `bailian` | `qwen3.5-plus` | 通义千问（推荐） |
| `bailian` | `qwen3.5-lite` | 快速模型 |
| `openai` | `gpt-4` | GPT-4 |
| `openai` | `gpt-3.5-turbo` | GPT-3.5 |
| `anthropic` | `claude-3-sonnet` | Claude 3 |

## 定时任务

### 基础配置

```json
{
  "cron": {
    "jobs": [
      {
        "name": "晨间简报",
        "schedule": {
          "kind": "cron",
          "expr": "0 8 * * *"
        },
        "payload": {
          "kind": "agentTurn",
          "message": "生成今日晨间简报"
        },
        "enabled": true
      }
    ]
  }
}
```

### 时间表类型

**Cron 表达式**：
```json
{
  "kind": "cron",
  "expr": "0 8 * * *"
}
```

**固定间隔**：
```json
{
  "kind": "every",
  "everyMs": 1800000
}
```

**指定时间**：
```json
{
  "kind": "at",
  "at": "2026-03-28T08:00:00+08:00"
}
```

### Cron 表达式示例

| 表达式 | 含义 |
|--------|------|
| `0 8 * * *` | 每天 8:00 |
| `0 */2 * * *` | 每 2 小时 |
| `*/30 * * * *` | 每 30 分钟 |
| `0 10 * * 1` | 每周一 10:00 |
| `0 9 1 * *` | 每月 1 号 9:00 |

## 安全配置

### 访问控制

```json
{
  "security": {
    "allowlist": ["+86138****", "@trusted_user"],
    "blocklist": ["+1234567890"],
    "rateLimit": {
      "requests": 100,
      "window": 3600
    }
  }
}
```

| 字段 | 类型 | 说明 |
|------|------|------|
| `allowlist` | array | 允许的用户 |
| `blocklist` | array | 禁止的用户 |
| `rateLimit.requests` | number | 请求次数 |
| `rateLimit.window` | number | 时间窗口 (秒) |

### API 密钥

```json
{
  "env": {
    "GITHUB_TOKEN": "ghp_xxx",
    "NOTION_TOKEN": "secret_xxx",
    "OPENWEATHER_API_KEY": "xxx"
  }
}
```

## 🔧 配置管理命令

```bash
# 查看配置
openclaw config get

# 编辑配置
openclaw config edit

# 验证配置
openclaw config validate

# 更新配置
openclaw config patch --path model.default --value qwen3.5-plus

# 重启应用
openclaw gateway restart
```
