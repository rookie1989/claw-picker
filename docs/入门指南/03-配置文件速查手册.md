# 配置文件速查手册

> OpenClaw 配置文件完整参考，包含所有常用配置项、默认值和示例。

---

## 📁 配置文件位置

| 系统 | 路径 |
|------|------|
| **Mac/Linux** | `~/.openclaw/config.yaml` |
| **Windows** | `C:\Users\你的用户名\.openclaw\config.yaml` |

## 🔧 完整配置结构

```yaml
# ==================== 模型提供商 ====================
providers:
  - name: anthropic              # 提供商名称
    apiKey: sk-ant-xxx           # API Key（必填）
    defaultModel: claude-sonnet-4-20250514  # 默认模型
    baseUrl: https://api.anthropic.com  # 可选：自定义 API 地址
    timeout: 60                  # 超时时间（秒）
    retry: 3                     # 重试次数
    fallback: false              # 是否作为备用
    
  - name: openai
    apiKey: sk-xxx
    defaultModel: gpt-4o
    fallback: true               # 主模型失败时使用
    
  - name: zhipu                  # 智谱 AI
    apiKey: xxx
    defaultModel: glm-4
    
  - name: aliyun                 # 通义千问
    apiKey: xxx
    defaultModel: qwen-max

# ==================== 渠道配置 ====================
channels:
  # Telegram
  - type: telegram
    token: 123456789:ABCdefxxx
    allowFrom:                   # 白名单（可选）
      - username: your_username
      - user_id: 123456789
    mentionOnly: false           # 只响应 @ 消息
    prefix: '/'                  # 命令前缀
    
  # WhatsApp
  - type: whatsapp
    sessionId: your_session_id   # 配对后自动生成
    
  # Discord
  - type: discord
    botToken: xxx
    guildId: "123456789"         # 服务器 ID
    allowFrom:
      - roleId: "987654321"      # 允许的角色
    
  # 飞书
  - type: feishu
    appId: cli_xxx
    appSecret: xxx
    encryptKey: xxx              # 可选：消息加密
    verificationToken: xxx       # 可选：事件验证
    
  # 钉钉
  - type: dingtalk
    clientId: xxx
    clientSecret: xxx
    agentId: "123456"

# ==================== 技能配置 ====================
skills:
  # 内置技能
  - name: browser                # 浏览器
    enabled: true
    config:
      headless: true             # 无头模式
      timeout: 30000             # 超时（毫秒）
      
  - name: search                 # 网络搜索
    enabled: true
    config:
      engine: brave              # brave/google/bing
      apiKey: xxx                # Brave Search API Key
      
  - name: files                  # 文件操作
    enabled: true
    config:
      allowedDirs:               # 允许访问的目录
        - ~/Documents
        - /tmp
        
  - name: calendar               # 日历
    enabled: true
    config:
      provider: google           # google/outlook/apple
      
  # 自定义技能
  - name: weather                # 天气查询
    script: ./skills/weather.js
    triggers:
      - keywords: ["天气", "气温"]
    config:
      defaultCity: Beijing
      
  - name: reminder               # 提醒
    script: ./skills/reminder.js
    triggers:
      - keywords: ["提醒", "定时"]

# ==================== 记忆配置 ====================
memory:
  enabled: true
  provider: local                # local/redis/postgres
  maxMessages: 100               # 保留消息数
  maxDays: 7                     # 保留天数
  config:
    dataPath: ~/.openclaw/memory

# ==================== 网关配置 ====================
gateway:
  port: 8080                     # HTTP 端口
  host: localhost                # 监听地址
  cors: true                     # 允许 CORS
  auth:                          # API 认证
    enabled: false
    token: your_api_token
    
# ==================== 日志配置 ====================
logging:
  level: info                    # debug/info/warn/error
  format: json                   # json/text
  output:
    - console
    - file
  file:
    path: ~/.openclaw/logs/gateway.log
    maxSize: 10MB                # 单个文件最大大小
    maxFiles: 5                  # 保留文件数

# ==================== 高级配置 ====================
# 多智能体路由
routing:
  enabled: false
  rules:
    - pattern: "代码.*"
      agent: developer-agent
    - pattern: "写作.*"
      agent: writer-agent

# 上下文管理
context:
  maxTokens: 100000              # 最大上下文长度
  compression: true              # 自动压缩
  summary: true                  # 自动摘要

# 安全配置
security:
  allowCommands:                 # 允许执行的命令
    - ls
    - cat
    - grep
  denyCommands:                  # 禁止执行的命令
    - rm -rf
    - sudo
  rateLimit:                     # 频率限制
    enabled: true
    maxRequests: 100
    windowMs: 60000              # 1 分钟

# 自定义变量
env:
  API_KEY: ${API_KEY}            # 从环境变量读取
  CUSTOM_VAR: value
```

## 🔑 常用配置项详解

### 模型提供商配置

#### Anthropic (Claude)

```yaml
providers:
  - name: anthropic
    apiKey: sk-ant-xxxxxxxxxxxx
    defaultModel: claude-sonnet-4-20250514
    # 可选模型：
    # - claude-sonnet-4-20250514 (推荐)
    # - claude-opus-4-20250514 (最强)
    # - claude-haiku-3 (最快)
```

#### OpenAI (GPT)

```yaml
providers:
  - name: openai
    apiKey: sk-xxxxxxxxxxxx
    defaultModel: gpt-4o
    # 可选模型：
    # - gpt-4o (推荐)
    # - gpt-4-turbo
    # - gpt-3.5-turbo (便宜)
```

#### 多模型 Failover

```yaml
providers:
  - name: anthropic
    apiKey: sk-ant-xxx
    defaultModel: claude-sonnet-4-20250514
    priority: 1                  # 优先级（数字越小优先级越高）
    
  - name: openai
    apiKey: sk-xxx
    defaultModel: gpt-4o
    priority: 2                  # 备用
    
  - name: zhipu
    apiKey: xxx
    defaultModel: glm-4
    priority: 3                  # 第三备用
```

### 渠道配置

#### Telegram 白名单

```yaml
channels:
  - type: telegram
    token: YOUR_TOKEN
    allowFrom:
      # 按用户名
      - username: your_username
      # 按用户 ID
      - user_id: 123456789
      # 按群组
      - chat_id: -1001234567890
```

#### 只响应 @ 消息

```yaml
channels:
  - type: telegram
    token: YOUR_TOKEN
    mentionOnly: true            # 只响应 @Bot 的消息
```

#### 命令前缀

```yaml
channels:
  - type: telegram
    token: YOUR_TOKEN
    prefix: '/'                  # 命令以 / 开头
    # 现在可以用 /help, /start 等命令
```

---

### 技能配置

#### 启用/禁用技能

```yaml
skills:
  - name: browser
    enabled: false               # 禁用浏览器
    
  - name: search
    enabled: true                # 启用搜索
```

#### 自定义技能触发器

```yaml
skills:
  - name: weather
    script: ./skills/weather.js
    triggers:
      # 关键词触发
      - keywords: ["天气", "气温", "多少度"]
      # 正则表达式触发
      - regex: "(\\d+) 天后天气"
      # 命令触发
      - command: "/weather"
```

---

### 记忆配置

#### 本地记忆

```yaml
memory:
  enabled: true
  provider: local
  maxMessages: 100               # 每个会话保留 100 条消息
  maxDays: 7                     # 保留 7 天
  config:
    dataPath: ~/.openclaw/memory
```

#### Redis 记忆（多实例共享）

```yaml
memory:
  enabled: true
  provider: redis
  config:
    host: localhost
    port: 6379
    password: your_password
    db: 0
```

## 🔐 安全最佳实践

### 使用环境变量存储敏感信息

```yaml
# ❌ 不要这样（明文密码）
providers:
  - name: anthropic
    apiKey: sk-ant-xxxxxxxxxxxx

# ✅ 应该这样（环境变量）
providers:
  - name: anthropic
    apiKey: ${ANTHROPIC_API_KEY}
```

```bash
# 在 ~/.bashrc 或 ~/.zshrc 中添加
export ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxx
```

### 配置命令白名单

```yaml
security:
  allowCommands:
    - ls
    - cat
    - grep
    - find
  denyCommands:
    - rm
    - sudo
    - chmod
    - chown
```

### 启用 API 认证

```yaml
gateway:
  auth:
    enabled: true
    token: ${API_TOKEN}
```
