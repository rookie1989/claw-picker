# 🌱 安装教程 - 详细版

10-15 分钟完成 OpenClaw 安装和配置。

## 📋 前置要求

- Node.js 18+ 
- npm 或 yarn
- 一个聊天应用账号（Telegram/WhatsApp/Discord/飞书等）
- 稳定的网络连接

## 🚀 步骤 1：安装 OpenClaw

### 方法 1：npm 全局安装（推荐）

```bash
npm install -g openclaw
```

验证安装：
```bash
openclaw --version
```

### 方法 2：源码安装

```bash
git clone https://github.com/openclaw/openclaw.git
cd openclaw
npm install
npm link
```

### 方法 3：Docker 安装

```bash
docker pull openclaw/gateway:latest
docker run -d \
  -v ~/.openclaw:/root/.openclaw \
  -p 3000:3000 \
  openclaw/gateway:latest
```

## 📝 步骤 2：初始化配置

运行交互式向导：

```bash
openclaw onboard
```

向导会引导你：
1. 选择要连接的聊天应用
2. 配置 API 凭证
3. 选择 AI 模型
4. 设置技能

### 手动配置

如果跳过向导，可手动编辑 `~/.openclaw/config.json`：

```json
{
  "plugins": {
    "entries": ["telegram", "weather"]
  },
  "model": {
    "default": "qwen3.5-plus"
  },
  "skills": {
    "entries": ["weather", "github"]
  }
}
```

## 🔧 步骤 3：连接聊天应用

### Telegram

1. 联系 [@BotFather](https://t.me/BotFather) 创建 Bot
2. 获取 Bot Token
3. 在配置中添加：

```json
{
  "plugins": {
    "telegram": {
      "token": "BOT_TOKEN_HERE"
    }
  }
}
```

### WhatsApp

1. 访问 [Meta for Developers](https://developers.facebook.com/)
2. 创建 WhatsApp Business App
3. 获取 Phone Number ID 和 Token
4. 配置 Webhook

### Discord

1. 访问 [Discord Developer Portal](https://discord.com/developers/applications)
2. 创建应用
3. 获取 Bot Token
4. 邀请 Bot 到服务器

### 飞书

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 创建企业自建应用
3. 获取 App ID 和 Secret
4. 配置权限和事件订阅

## 🤖 步骤 4：配置 AI 模型

### 使用内置模型

```json
{
  "model": {
    "default": "qwen3.5-plus"
  }
}
```

### 使用自有模型

```json
{
  "model": {
    "provider": "openai",
    "apiKey": "sk-xxx",
    "default": "gpt-4"
  }
}
```

### 多模型配置

```json
{
  "model": {
    "default": "qwen3.5-plus",
    "fallback": "qwen3.5-lite",
    "providers": {
      "openai": { "apiKey": "sk-xxx" },
      "anthropic": { "apiKey": "sk-ant-xxx" }
    }
  }
}
```

## 🛠️ 步骤 5：启用技能

### 内置技能

```json
{
  "skills": {
    "entries": [
      "weather",
      "github",
      "feishu-bitable",
      "feishu-calendar"
    ]
  }
}
```

### 自定义技能

将技能放在 `~/.openclaw/skills/` 目录下：

```
~/.openclaw/skills/
├── my-weather-skill/
│   └── SKILL.md
└── my-custom-skill/
    └── SKILL.md
```

## ▶️ 步骤 6：启动网关

```bash
openclaw gateway start
```

查看状态：
```bash
openclaw gateway status
```

查看日志：
```bash
openclaw gateway logs
```

## ✅ 步骤 7：测试连接

发送消息给你的 Bot：

```
你好
```

应该收到 AI 回复。

## 🐛 常见问题

### Q: 安装失败？

**A:** 检查 Node.js 版本：
```bash
node --version  # 需要 18+
npm --version
```

升级 Node.js：
```bash
nvm install 18
nvm use 18
```

### Q: 网关启动失败？

**A:** 查看详细错误：
```bash
openclaw gateway start --verbose
```

常见原因：
- 端口被占用
- 配置文件格式错误
- 缺少必要凭证

### Q: 消息无响应？

**A:** 检查：
1. 网关是否运行
2. 聊天应用配置是否正确
3. AI 模型凭证是否有效
4. 网络连接是否正常

```bash
# 查看日志
openclaw logs --level debug

# 测试 API
curl -I https://api.example.com
```

### Q: 如何更新 OpenClaw？

**A:** 
```bash
npm update -g openclaw
openclaw gateway restart
```
